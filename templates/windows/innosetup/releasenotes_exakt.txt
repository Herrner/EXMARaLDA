RELEASE NOTES für den EXMARaLDA Partitur-Editor Version 1.3.4 (30-Juni-2007)

Version 1.3.4. behebt einige Fehler aus Version 1.3.4. und beinhaltet zusätzliche Funktionalität.

Ab Version 1.3.3. unterstützt EXMARaLDA nur noch Java-Versionen ab 1.4. 
Dies bedeutet, dass neue Versionen des Partitur-Editors auf MAC OS X, 
Version 10.2. (Jaguar) und 10.1. (Puma) nicht lauffähig sein werden, weil es für diese Betriebssysteme keine geeignete Java-Version gibt. 
Falls Sie auf eine Version des Partitur-Editors angewiesen sind, die unter Java 1.3. lauffähig ist, schreiben Sie mir bitte eine e-mail.


1. Zusätzliche Funktionalität
=============================

Verbesserte Audio- und Video-Unterstützung
-------------------

Durch eine Integration der FOBS4JMF und Quicktime-Bibliotheken gibt es jetzt eine verbesserte Unterstützung für das Playback von Audio- und Video-Daten.
Insbesondere können modernere Video-Codecs (Divx, xvid, mpeg-4) verwendet werden.
Genaueres entnehmen Sie bitte dem Dokument "Audio and Video support in EXMARaLDA"
der das JLayer-Framework verwendet werden. Letzteres macht es möglich, auch auf dem MAC MP3-Dateien abzuspielen. Welches Framework verwendet wird, wird über „Edit > Preferences > Media“ festgelegt.

Partitur-Ausgabe
----------------

Ein Ausschnitt aus einer Partitur kann jetzt über den Copy-Befehl einfach in die Zwischenablage gelegt und in einem RTF-fähigen Programm eingefügt werden.
