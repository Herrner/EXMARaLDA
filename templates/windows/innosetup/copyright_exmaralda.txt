EXMARaLDA

This EXMARaLDA distribution contains the following software tools:

Partitur Editor Version 1.6
Corpus Manager Version 2.0.1488
EXAKT Search tool Version 1.3

The tools are developed by Thomas Schmidt, Kai Wörner and Hanna Hedeland.

German localisations by Annette Schnieders
Turkish localisations by Tülay Selcuk

Czech localisations by Petr Porizka
Spanish localisations by Silke Merkel
Chinese localisations by Fan Junjun
