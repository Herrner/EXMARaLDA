*Anmerkung zu Mac OS und Java 6

Für Mac OS 10.5 gibt es im Augenblick keine offizielle Java 6-Laufzeitumgebung, die für die aktuellsten (>1.0) Versionen von Coma benötigt wird. Wenn Sie einen Intel-Mac mit OS 10.5 haben, so können sie eine Vorabversion von Java 6 bei der Apple Connect-Webseite (nach Anmeldung) herunterladen.
